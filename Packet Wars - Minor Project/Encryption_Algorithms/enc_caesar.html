Encryption Algorithms

Caesar Cipher:
  • It is a monoalphabetic substitution cipher and one of the easiest encryption method till date.
  • In this method, each character is shifted by as many times as the value of key.
  • As it is just substituition by shifting the characters, it is very easy to break the ciphertext.
  • Using all the 26 alphabet as keys, one of them has to give the correct plaintext making it very insecure.
